# Svelte + Vite

## Commands

### create project
- git clone https://github.com/jaoneol/Frontend.git
- cd pjt01-movieinfo
- npm install
- npm run dev

### docker build & run container
- docker build -t pjt01-movieinfo .
- docker run -d -p 8080:80 pjt01-movieinfo
